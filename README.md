## ◈ Featured Projects

<details>
<summary><strong>◉ Lexis — Production-Grade Multi-Chat RAG SaaS</strong></summary>
<br/>

> A full-stack, production-ready Retrieval-Augmented Generation SaaS platform with multi-workspace support, real-time web search augmentation, persistent vector storage per user, and enterprise-grade authentication. Designed as a deployable AI knowledge management product — not a prototype.

<div align="center">

| Dimension | Details |
|-----------|---------|
| **Stack** | FastAPI · React · Vite · Clerk · Neon Postgres · ChromaDB · Gemini Embeddings · Groq · Tavily · SQLAlchemy · Alembic |
| **Scale** | Multi-user, multi-workspace, per-collection vector isolation |
| **Performance** | Async Python throughout, Groq inference for sub-second LLM response, streaming support |
| **Security** | Clerk JWT authentication, per-user data scoping, environment-isolated secrets |
| **Impact** | Featured portfolio centerpiece — demonstrates end-to-end AI product engineering |
| **Repository** | [![GitHub](https://img.shields.io/badge/GitHub-GhananilShirpurkar%2FLexis-6d28d9?style=flat-square&logo=github)](https://github.com/GhananilShirpurkar/Lexis) |

</div>

Lexis implements a complete RAG pipeline: documents are chunked, embedded via Gemini Embeddings, and stored in ChromaDB with per-user/collection scoping. At query time, the system retrieves semantically relevant context, optionally augments with live Tavily web search, and synthesizes a grounded response via Groq. Multi-chat architecture allows parallel conversation threads with isolated memory and context per session.

</details>

<details>
<summary><strong>◉ MediSync — Multi-Agent Pharmacy Automation System</strong></summary>
<br/>

> An enterprise-grade, multi-agent pharmacy automation platform built on LangGraph, designed to streamline prescription intake, drug interaction checking, inventory management, and patient communication — all through coordinated AI agents with a production FastAPI backend and React frontend.

<div align="center">

| Dimension | Details |
|-----------|---------|
| **Stack** | LangGraph · FastAPI · React · EasyOCR · Whisper · PyTorch · Groq |
| **Scale** | Multi-agent graph with 5+ specialized agents, event-driven inter-agent messaging |
| **Performance** | Parallel agent execution via LangGraph conditional edges, async task dispatch |
| **Security** | Role-based access, isolated agent state, sandboxed OCR processing |
| **Impact** | Primary hackathon showcase — demonstrates production multi-agent orchestration |
| **Repository** | [![GitHub](https://img.shields.io/badge/GitHub-GhananilShirpurkar%2FMediSync-4f46e5?style=flat-square&logo=github)](https://github.com/GhananilShirpurkar/MediSync) |

</div>

MediSync uses a supervisor-worker agent architecture: a coordinator agent routes incoming pharmacy tasks to specialized sub-agents (prescription parser, drug checker, inventory agent, notification agent). Each agent maintains isolated state within the LangGraph graph while sharing a structured output schema, enabling reliable cross-agent data handoff without prompt chaining brittleness.

</details>

<details>
<summary><strong>◉ SafeGuardAI — On-Device Android Scam Detection</strong></summary>
<br/>

> A personal Android application for real-time SMS and call scam detection using an on-device TFLite classification model combined with Gemini API for contextual threat explanation — designed for low-resource Android devices with no server dependency for core inference.

<div align="center">

| Dimension | Details |
|-----------|---------|
| **Stack** | Android · TFLite · Gemini API · Python (ML pipeline) · scikit-learn |
| **Scale** | On-device inference — no server round-trip for classification |
| **Performance** | TFLite INT8 quantized model, sub-20ms inference on mid-range Android |
| **Security** | Privacy-first: no message data leaves the device during classification |
| **Impact** | Demonstrates full ML pipeline ownership — data, training, conversion, deployment |
| **Repository** | [![GitHub](https://img.shields.io/badge/GitHub-GhananilShirpurkar%2FSafeGuard-AI-7c3aed?style=flat-square&logo=github)](https://github.com/GhananilShirpurkar/SafeGuardAI) |

</div>

SafeGuardAI is a solo project covering the complete ML pipeline end-to-end: data preprocessing, scikit-learn model training, TFLite conversion with INT8 quantization, and Gemini API integration for human-readable threat explanation. The on-device inference architecture eliminates latency and privacy concerns inherent in cloud-dependent scam detection.

</details>
