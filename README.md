## About

IT student at SGGSIE&T (Batch of '28) focused on one thing: building agentic AI systems that actually work.

Most of my projects sit at the intersection of LLM orchestration and full-stack engineering — multi-agent pipelines, real-time backends, usable interfaces. I'm not interested in models that stay in notebooks.

Currently deepening on: **LangGraph · RAG pipelines · LLM evaluation · agentic system design**

## 🔨 Projects

**[MediSync](https://github.com/GhananilShirpurkar/medisync)** — Multi-agent pharmacy automation system
Built in 48 hours at a hackathon. LangGraph-orchestrated agents handle medication ordering, OCR prescription parsing, behavioral risk scoring, proactive refill prediction, and WhatsApp order notifications.
`LangGraph` `FastAPI` `Gemini Vision` `Twilio` `React` `SQLite`

**[Fake News Detector](https://github.com/GhananilShirpurkar/fake-news-detector)** — NLP-based news classification
TF-IDF + scikit-learn classifier with a full desktop GUI — explainability panel, readability meter, keyword highlighter, animated verdict reveal.
`Python` `scikit-learn` `CustomTkinter` `NLP`
